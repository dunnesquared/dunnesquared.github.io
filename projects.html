Cinescoutüéû (v1.0.1)

My latest project!

Cinescout is a Flask-based, mobile-responsive website where you can learn more about any film or person in the world of cinema. All of Cinescout's data is fetched in real-time from The Movie Database and The New York Times.

With Cinescout you can

  • Read a brief synopsis of a film from The Movie Database, as well as a review summary from The New York Times.
  • Discover the filmography of your favourite cast or crew member.
  • Create an account to add movies to a personal list for later reference.
  • Browse through a list of critically-acclaimed films from The Criterion Collection.

A online demo of the Cinescout exists, but I've walled it off from the public for now because it's not production-ready, so to speak. I hope to have some version of app available for general use one day. There are several, non-trivial issues that I to solve first, however (e.g. migration to a robust database; data backups; API request limits, etc.).

In the mean time, I have created an award-winning screencast of Cinescout you can enjoy. I hear it works wonders for insomnia :-p. You can explore the code on github too.

If you would like to try the demo and we're on friendly terms, please contact me :-).

Tech used:
Python 3.6.1, Flask, JavaScript ES6, requests, SQLAlchemy, Flask-Login, Flask-WTF, unittest, SQLite, Bootstrap 4.5, Bash, Ajax, DataTables, fuzzywuzzy

Sentence Cow is a Flask-based web tool that extracts sentences from a text, or at least tries to as best as possible (it's not a bad word- and character- counter either). It doesn't use any NLP algorithms to detect sentences‚Äìjust a lot regular expressions and condition-checking.

As intimated, Sentence Cow doesn't always get it right, sometimes detecting a sentence where it shouldn't or not detecting one where it should. No problem‚Äîusers can fix either of these problems by using the provided 'split' or 'merge' tools to divide or join strings, respectively.
